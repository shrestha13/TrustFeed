How retractions work

① Publisher identity verified — only the original submitter can retract

② Retraction document signed with same Ed25519 key that signed the IOC

③ Signed retraction saved to retraction log (SQLite)

④ IOC marked as retracted in IOC store

⑤ Any future verification of this IOC ID is blocked

Retractions provide non-repudiation — the signature proves which publisher retracted and when. This is the cryptographic equivalent of a signed recall notice.
